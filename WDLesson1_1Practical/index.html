Web Technologies

HTML

HTML stands for Hyper Text Markup Language. Which is a coding language to dispaly content on a webpage. It is not a programming language.

CSS

CSS stands for Cascading Style Sheets. It's a style sheet language used in HTML to customize how web pages look. And so use it to change size, color, fonts of the words or change other things.

JavaScript

JavaScript is a high-level, multi-paradigm programming language. Mainly known for allowing interactive and dynamic content on websites. It is one of the core technologies of the World Wide Web, alongside HTML and CSS.
